# Basic-Quiz-App
A simple and responsive Quiz App built using HTML, CSS, and JavaScript. It dynamically loads questions, tracks user answers, and displays the final score at the end. Designed with a clean UI and basic responsiveness for different screen sizes.



## 📌 Overview

This is a simple and responsive Quiz Application built using HTML, CSS, and JavaScript. The app presents multiple-choice questions, tracks user answers, and displays the final score at the end.

## 🚀 Features

- Dynamic question loading
- Multiple-choice options
- Score calculation
- Final result display
- Responsive design using media queries
- Clean and minimal UI

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 📂 Project Structure

- "index.html" – Main structure of the app
- "style.css" – Styling and responsiveness
- "app.js" – Quiz logic and functionality

## 📱 Responsiveness

The app is designed to work across different screen sizes using media queries for tablets and mobile devices.

## ▶️ How to Run

1. Download or clone the repository
2. Open "index.html" in your browser
3. Start the quiz and enjoy
